
### Assistants API with Code interpreter & Retrieval Tool을 가지고 Image를 만들기
- 코드 해석기(Code Interpreter)를 사용하면 Assistants API를 사용하여 샌드박스 실행 환경에서 Python 코드를 작성하고 실행할 수 있습니다.
